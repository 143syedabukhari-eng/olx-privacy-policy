This Privacy Policy describes how the OLX Data Extractor & Lead Scraper ("the Extension") handles user information.

1. Data Collection and Usage

The Extension is a tool designed to help users extract marketplace listing data for research purposes. All data extraction and processing occur locally on the user's computer.

  • No Personal Data Collection: We do not collect, store, or transmit any personal information about you to external servers.
  • Local Storage: Scraped data (such as listing titles, prices, and phone numbers) is stored locally in your browser's storage only.
  • User Control: You can clear all stored data at any time by clicking the "Clear Data" button within the extension popup.
